Эта программа представляет собой простой телефонный справочник с консольным интерфейсом. Она предоставляет пользователю несколько основных функций для работы с записями в телефонной книге. Вот подробное описание функционала:

Вывод записей: Пользователь может просматривать записи в телефонной книге. Записи выводятся постранично, с возможностью перехода к следующей или предыдущей странице. Пользователь может выбирать, сколько записей отображать на одной странице.

Добавление записи: Позволяет пользователю добавить новую запись в телефонную книгу. Пользователь вводит данные о новой записи, такие как фамилия, имя, отчество, название организации и телефоны. После добавления записи программа сохраняет изменения в файле JSON.

Редактирование записи: Позволяет пользователю изменить существующую запись в телефонной книге. Пользователь указывает номер записи, которую он хочет изменить, и поле, которое нужно отредактировать. После ввода нового значения программа сохраняет изменения в файле JSON.

Поиск записей: Позволяет пользователю искать записи в телефонной книге по одной или нескольким характеристикам. Пользователь вводит характеристики для поиска, и программа выводит все записи, которые соответствуют введенным характеристикам.

Загрузка файла JSON: Позволяет пользователю загрузить данные из другого файла JSON. Пользователь вводит имя файла, и программа проверяет его наличие. При успешной загрузке программа выводит сообщение о успешной загрузке и использует новые данные для работы.

Выход из программы: Пользователь может завершить работу программы, сохранив текущие данные в файле JSON.

Программа также обрабатывает некорректный ввод пользователя и предоставляет соответствующие сообщения об ошибке.
